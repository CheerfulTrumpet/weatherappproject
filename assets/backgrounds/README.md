# City background assets

The app now loads these bundled files first before falling back to the generated backgrounds in `src/background_renderer.py`.

## Replaceable files
- `new_york.png`
- `london.png`
- `paris.png`
- `tokyo.png`
- `sydney.png`
- `dubai.png`
- `singapore.png`

## Best results
- Use `1920x1080` or larger
- Choose a clear skyline or landmark shot for each city
- Keep the main landmark near the center or upper-middle area
- Avoid very bright all-white images so the dashboard text stays readable

You can replace any PNG with your own city artwork or photo using the same filename.
